# Interactive Quiz Application

JavaScript-powered quiz/trivia application with dynamic question loading.

## Features
- **Multiple Choice**: Question with answer options
- **Dynamic Loading**: Questions loaded programmatically
- **Navigation**: Next button progression
- **Answer Selection**: Button-based answer choice
- **Score Tracking**: Result calculation
- **Clean Interface**: User-friendly design

## Technology Stack
- HTML5
- CSS3
- JavaScript
- DOM manipulation

## Components
- Question display
- Answer buttons
- Navigation controls
- Score calculator

## Use Cases
- Educational quizzes
- Training assessments
- Trivia games
- Knowledge testing
- E-learning platforms
